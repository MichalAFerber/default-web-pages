# Tinyproxy - default error page

The error page Tinyproxy serves (400/403/404/502/503, etc.) - a tiny XHTML page that ends
in "Generated by tinyproxy." `default.html` is the genuine template; Tinyproxy fills the
`{errno}`, `{cause}`, `{detail}`, `{package}`, and `{website}` placeholders at runtime.

## Provenance

Byte-exact from tinyproxy/tinyproxy, `data/templates/default.html`:
<https://github.com/tinyproxy/tinyproxy/blob/master/data/templates/default.html>.

(`stats.html` and `debug.html` are the other two templates in that folder.)
