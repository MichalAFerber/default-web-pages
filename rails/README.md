# Ruby on Rails - "Yay! You're on Rails!"

The welcome page a fresh Rails app shows at `/`: the big "Yay! You're on Rails!" with the
Rails logo.

`index.html.erb` is the genuine ERB template Rails ships for the welcome controller (the
inline Rails logo SVG is part of it).

## Provenance

Byte-exact from rails/rails, `railties/lib/rails/templates/rails/welcome/index.html.erb`:
<https://github.com/rails/rails/blob/main/railties/lib/rails/templates/rails/welcome/index.html.erb>.
