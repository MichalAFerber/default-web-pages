# Laravel - welcome page

The default page a new Laravel app serves at `/`: the dark, Tailwind-styled landing page
with the Laravel logo and links to the docs, Laracasts, the ecosystem, and so on.

`welcome.blade.php` is the genuine Blade template Laravel scaffolds into
`resources/views/`. It is a Blade/Tailwind source file - it renders inside Laravel, not
as standalone HTML.

## Provenance

Byte-exact from laravel/laravel, `resources/views/welcome.blade.php`:
<https://github.com/laravel/laravel/blob/master/resources/views/welcome.blade.php>.
